Release notes de la versió 1.4.4 de Canigó

Canvis desenvolupats a Canigó 1.4.4


  • Canvi CAN-1218: Select depenent de dos select
    Context: Capacitat per poder fer filtrar el contingut d'un select depenent de més d'un select
    Servei afectat: openframe-service-web
    Solució: Solució implementada en la CAN-1222

  • Canvi CAN-1222: Select depenent de dos select o més selects
    Context: Capacitat per poder fer filtrar el contingut d'un select depenent de més d'un select
    Servei afectat: openframe-service-web
    Contingut estàtic afectat: openFrame-ajaxtags-select.js, ajaxtags-1.1.js
    Solució: Canvi en el servei de presentació de tags per tal d'oferir la possibilitat que un select pogui dependre de dos o més selects.
    Incidències Resoltes a Canigó 1.4.4


  • Incidència CAN-1213: Refresc d'una select depenent d'una select depenent
    Context: Una select depenent d'una altre select depenent d'un 3er select no funciona
    Connector afectat: openframe-service-web
    Contingut estàtic afectat: openFrame-ajaxtags-select.js
    Solució: Modificació del servei de presentació de tags per tal de que els selects depenents poguin dependre de N selects depenents.
